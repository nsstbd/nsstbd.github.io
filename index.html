Workshop on New storage system and technology for Big Data
BDSE Workshop

Introduction

In recent years data storage has emerged as important research fields driven by the demand for scalable structures and technologies to satisfy the growing needs of massive data management and processing. Big Data challenges storage systems with more capacity, scalability and efficient accessibility, while traditional storage architectures just werenâ€™t designed to handle this scale. This workshop intends to bring together researchers and practitioners from academia and industry to discuss cutting edge research on new storage systems and technology for big data, such that the interactions among the experts of these areas can be promoted and new interdisciplinary technologies can be developed.

Research topics included in the workshop

Topics of interest of this workshop include but are not limited to:

  • Non-volatile memory-based storage systems
  • Applications of non-volatile memories to big data
  • Hybrid storage systems
  • Solid state drive (SSD)
  • Data deduplication
  • Key-value and NoSQL storage
  • Storage reliability, availability and security
  • Cache and memory systems
  • Memory computing
  • Operating system and file system designs for non-volatile memories
  • Distributed and parallel I/O
  • Energy aware storage
  • Storage performance and scalability
  • Storage virtualization and cloud storage
  • Storage management for big data
